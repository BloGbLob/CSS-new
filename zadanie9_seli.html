  3. Gdzie wstawiany jest styl zewnętrzny?


    Wstawiany w sekcji <head> za pomocą znacznika <link>. Może występować wielokrotnie.

  4. Najprostsza forma wstawiania stylu zewnętrznego:

        <link rel="stylesheet" href="style.css">

  5. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    Do jednego dokumentu HTML można dołączyć wiele arkuszy stylów. Należy pamiętać, że jeśli w różnych plikach CSS są sprzeczne deklaracje, ważniejszy będzie ten załadowany później.

  6. Co się dzieje w przypadku konfliktów?

    Jeśli ten sam element jest stylizowany w różnych arkuszach, ostatnio załadowany styl nadpisuje poprzednie. Można użyć !important, by nadać priorytet określonej regule.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    Pliki CSS można tworzyć w prostych edytorach tekstowych jak Notatnik czy Notepad++, zapisując je z rozszerzeniem .css.

  8. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    Można skorzystać z edytorów takich jak Visual Studio Code, Sublime Text czy Brackets, które ułatwiają pracę z CSS.

  9. Jakie rozszerzenie posiadają plik będący zewnętrznym arkuszem styli?

    Pliki arkuszy styli zawsze posiadają rozszerzenie .css.

  10. Jakie są zalety zewnętrznego arkusza styli?

    • Oddzielenie struktury dokumentu HTML od jego wyglądu, co ułatwia zarządzanie kodem.
    • Możliwość wielokrotnego wykorzystania arkusza stylów w różnych dokumentach HTML, co zwiększa spójność wyglądu.

  11. Uwagi na temat pisania w CSS

    • Należy unikać nadmiernego używania !important, ponieważ utrudnia to zarządzanie stylami.
    • Dobrą praktyką jest organizowanie kodu w czytelny sposób, np. grupując podobne style.
