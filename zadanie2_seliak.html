Definicja CSS - Kaskadowych Arkuszy Styli

CSS - Kaskadowe Arkusze Stylów w (z ang. Cascading Style Sheets) jest to specjalny język stworzony w celu elastycznego zarządzania sposobem formatowania (wyglądem) elementów znajdujących się na stronach WWW.
